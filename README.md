# Laku - WhatsApp POS (Warung Pak Haji Edition)

A single-tenant WhatsApp Point of Sale system built with Next.js, Prisma, and WhatsApp Business API.

## Features
- **Digital Menu**: Public web view of products.
- **WhatsApp Bot**: Auto-reply with menu and order handling.
- **Admin Dashboard**: Manage products, categories, and view live orders.
- **Single Tenant**: Designed to be deployed per-merchant (e.g., one for Pak Haji, one for Cafe B).

## Tech Stack
- Next.js 15 (App Router)
- PostgreSQL + Prisma
- Tailwind CSS + Shadcn/ui
- WhatsApp Cloud API
